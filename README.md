# 🌿 TIL (Today I Learned) & Knowledge Base

AI/ML 엔지니어 성장 기록 저장소.

## 📂 Directory Structure & Content Guide

```text
.
├── 00_Basics
├── 01_Lectures
├── 02_Papers
├── 03_Tech_Stack
├── 04_Trends
├── 05_Coding_Test
├── 06_Kaggle_Practice
├── README.md
└── 취업계획.md
```

### 00_Basics (기초 체력)
- `CS_Fundamentals/`: 운영체제, 네트워크, 데이터베이스 등 전산학 기초 이론 정리함.
- `Dev_English.md`: 기술 문서/아티클 기반 개발 영어 표현 및 용어 아카이빙함.

### 01_Lectures (입력: 강의)
- `[Course_Name]/`: 강의별 개념 요약, 실습 코드, 재해석 노트 기록함.

### 02_Papers (심화: 논문 리뷰)
- `*.md`: 논문 핵심 아이디어, 아키텍처, 주요 실험 결과 요약함.
- `images/`: 논문 figure, 구조도, 표 이미지 아카이빙함.
- `README.md`: 논문 요약 테이블 및 최신 기록 관리함.

### 03_Tech_Stack (도구: 프레임워크 & 인프라)
- `Frameworks/`: PyTorch, TensorFlow, Scikit-learn 활용 패턴 정리함.
- `Infra_Ops/`: Docker, Kubernetes, MLOps(MLflow 등) 운영/배포 지식 정리함.
- `Database/`: PostgreSQL 등 DB 설계, 쿼리 최적화 경험 정리함.
- `Troubleshooting/`: 에러 로그, 원인 분석, 해결 절차 아카이빙함.

### 04_Trends (시야: 이슈 및 뉴스)
- `YYYY-MM_News.md`: 월별 모델/산업 동향, 기사 요약, 인사이트 기록함.

### 05_Coding_Test (논리: 알고리즘)
- `Platform_Name/`: 플랫폼별 문제 풀이, 복잡도 분석, 시행착오 기록함.

### 06_Kaggle_Practice (실전: 데이터 분석)
- `Competition_Name/`: 대회별 EDA, 피처 엔지니어링, 모델 실험 로그 정리함.
